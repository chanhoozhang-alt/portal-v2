# 项目框架说明

## 项目概况

| 项目 | 说明 |
|------|------|
| 项目名称 | FF59.1250001_njAppManSysPc |
| Maven GAV | `nj-appman-parent` |
| 基础包名 | `com.cmbchina.njappman` |
| 项目定位 | 南京分行应用管理系统 |

## 整体目录结构

```
nj-appman-parent/
├── common/                    # 公共模块
├── services/                  # 服务模块（聚合目录）
│   ├── ai-service/            # AI 服务
│   ├── console-service/       # 管理后台服务
│   ├── portal-service/        # 门户前台服务
│   └── server-service/        # 认证/基础服务
├── docs/                      # 项目文档
├── BuildScript/               # 构建脚本
├── src/                       # 根模块资源（Dockerfile 等）
└── pom.xml                    # 父 POM
```

## 模块说明

### 1. common — 公共模块

所有服务共享的基础能力，被其他服务以 Maven 依赖方式引用。

```
com.cmbchina.njappman.common
├── cache/           # 缓存管理（Redis 键定义、缓存读写/失效）
├── config/          # 通用配置（MyBatis 审计填充、序列化等）
├── context/         # 用户上下文（ThreadLocal 持有当前登录信息）
├── core.config/     # 核心配置（Spring Bean 装配等）
├── exception/       # 统一异常定义与全局异常处理器
├── model/           # 数据模型（Entity、DTO、VO、枚举、常量）
├── security/        # 安全机制（认证拦截、权限校验、内部 Token）
└── util/            # 工具类
```

### 2. services — 服务模块

采用聚合目录管理，每个子服务是独立的 Spring Boot 应用，可独立部署。

| 服务 | 职责 |
|------|------|
| **ai-service** | AI 智能服务（智能问答、辅助决策等） |
| **console-service** | 管理后台（系统管理员操作入口） |
| **portal-service** | 门户前台（普通用户操作入口） |
| **server-service** | 认证与基础服务（登录、Token 管理、用户同步） |

### 3. docs — 项目文档

| 文件 | 内容 |
|------|------|
| `sql.txt` | 数据库建表脚本 |
| `南京分行应用管理系统-基础需求文档.md` | 业务需求描述 |
| `接口文档.md` | 前后端接口契约 |
| `权限控制设计方案.md` | 权限模型与校验流程设计 |
| `项目框架说明.md` | 本文档 |

### 4. BuildScript — 构建脚本

存放 Maven 构建脚本、CI/CD 配置等。

### 5. src — 根模块资源

| 文件 | 用途 |
|------|------|
| `Dockerfile` | 容器化部署镜像定义 |
| `pom.xml` | 根模块 POM（如有） |
| `README.md` | 项目说明 |
| `.gitignore` | Git 忽略规则 |

```
                    ┌──────────────┐
                    │   Nginx(80)  │
                    └──────┬───────┘
           ┌───────────────┼───────────────┐
           ▼               ▼               ▼
   ┌──────────────┐ ┌──────────────┐ ┌──────────────┐
   │portal-service│ │console-service│ │  ai-service  │
   │   (门户)     │ │  (管理后台)   │ │  (AI 服务)   │
   └──────┬───────┘ └──────┬───────┘ └──────┬───────┘
          │                │                │
          └────────────────┼────────────────┘
                           ▼ Feign + 内部 Token
                  ┌──────────────────┐
                  │  server-service  │

                  └──────────────────┘
```

## 技术栈推断

基于项目结构和包命名推断：

| 技术 | 用途 |
|------|------|
| Java 8 + Spring Boot | 基础框架 |
| Spring Cloud + Nacos | 服务注册/配置中心 |
| MyBatis-Plus | ORM |
| Redis | 缓存/Token 存储 |
| Hutool | 工具库 |
| Lombok | 代码简化 |
